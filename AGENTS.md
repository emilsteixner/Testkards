# Testbild / Test Card Generator

## Overview
After Effects project for generating test pattern cards ("Testbild" in German) at multiple frame rates. Each composition targets a specific FPS value and includes shape layers, audio test tones, and text overlays.

## Project Structure

| File / Directory | Description |
| ---------------- | ----------- |
| `testbild.aep`   | Main After Effects project (binary RIFX format) |
| `testbild.aepx`  | XML-flavored export of the project for external parsing |
| `src/`           | Source assets: Illustrator grid, logo PNG, 600Hz / 700Hz wav files |
| `testbild_AME/`  | Render queue outputs produced by Adobe Media Encoder |
| `venv/`          | Python virtual environment (ignored) |

## Compositions
- **24FPS**, **25FPS**, **30FPS**, **50FPS**, **60FPS** — each with shape layers, text, two audio tracks
- **Unbenannt-1** — single layer composition (embedded logo / artwork)

## Technical Notes
- Parsed with `py-aep` (Forticheprod) for programmatic inspection
- Audio tracks: `600Hz.wav` and `700Hz.wav` serve as audible tone references
- Text layers display the target frame rate numerically (e.g., "24FPS")
- `src/grid.ai` provides the vector test pattern used in the shape layers

## Dependencies
- Adobe After Effects (latest CC)
- (Optional) `py-aep` for Python-based AEP analysis
